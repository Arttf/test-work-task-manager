# Инструкция по запуску

Перед запуском бэкенда укажите путь к SQLite-файлу в `back/.env.local`:

```env
DATABASE_URL=sqlite:///C:/Users/.../File.db
```

Важно:
- Используйте абсолютный путь к вашему `.db` файлу.
- Формат должен быть именно `sqlite:///...`.
